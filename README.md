**Turn your boring Windows desktop into a creative canvas.**

*Custom Windows Widget* is a lightweight, native desktop engine that lets you pin aesthetic image widgets—like Pinterest boards, reference photos, or mood boards—directly to your wallpaper. Designed for minimal CPU impact and maximum aesthetic.

---

## ✨ Features

* **🖼️ Native Widgets:** Place images directly on your desktop from local files or Web URLs.
* **⚡ Zero Lag:** Multi-threaded architecture ensures instant startup and 0% lag while gaming or working.
* **🛠️ Interactive Edit Mode:** Drag, pan, resize, and arrange your widgets exactly how you want them.
* **🔒 Lock Layout:** One-click locking mechanism makes widgets part of your background (click-through).
* **🔄 Auto-Updates:** Built-in system automatically detects, downloads, and installs new features.
* **👻 Stealth Mode:** Runs quietly in the System Tray with no taskbar clutter.

## 📥 Download & Install

1.  Go to the **[Releases Page](https://github.com/designswithharshit/Custom-Windows-Widget/releases/latest)**.
2.  Download the latest installer: `WinWidget_Setup_v1.0.exe`.
3.  Run the installer. The app will launch automatically to the System Tray.

> **Note:** On the first run, you might see a Windows SmartScreen warning because this app is not yet signed with a paid certificate. Click **"More Info" -> "Run Anyway"**.

## 🎮 User Guide

### 1. Getting Started
* Once installed, check your **System Tray** (bottom right near the clock) for the **Red Dot Icon** (or App Icon).
* Right-click the icon to open the **Control Menu**.

### 2. Adding Widgets
* **Right-Click Tray Icon** → Select **"Add Widget"**.
* A default widget will appear on your screen.

### 3. Customizing (Edit Mode)
* **Right-Click Tray Icon** → Select **"Start Editing Layout"**.
* **Move:** Left-click and drag any widget.
* **Resize:** Drag the bottom-right corner.
* **Pan Image:** Hold `Shift` + Drag inside the widget to move the image within the frame.
* **Change Image:** Right-click any widget -> **"Source Image"** -> Paste URL or Choose Local File.
* **Lock It:** Right-click Tray -> **"Finish Editing"**.
